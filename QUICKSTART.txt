VPA (Virtual Personal Assistant)
Linux Deployment - Quick Start Guide
================================================

📦 WHAT YOU HAVE:
-----------------
✓ Modular VPA architecture (main.py refactored from 2090 to ~300 lines)
✓ All functionality split into:
  - games/   (TicTacToe, Connect Four, Guess the Word, Dictionary)
  - tools/   (timer, alarm, email, bangla translator, general search)
  - media/   (stock tracker, crypto tracker, news, camera drawing)
  - utils/   (launcher, TTS, platform helpers)
✓ Complete systemd service setup files
✓ Verification and setup scripts

🚀 QUICK START:
--------------
1. Test it manually first:
   ./run_vpa.sh

2. Run verification:
   python3 verify_setup.py

3. Install as a service:
   ./setup_linux_service.sh
   (Follow the printed instructions)

4. Enable and start:
   sudo systemctl enable vpa.service
   sudo systemctl start vpa.service

5. Check it's running:
   sudo systemctl status vpa.service

🔊 SETUP CHECKLIST:
------------------
□ Install dependencies: pip install -r requirements.txt
□ Install system packages (Ubuntu/Debian):
  sudo apt install python3-pyaudio portaudio19-dev ffmpeg
□ Add yourself to audio group: sudo usermod -a -G audio $USER
□ Add yourself to video group: sudo usermod -a -G video $USER
□ (Optional) Get alarm.wav file: ./how_to_get_alarm_sound.sh
□ Log out and back in for group changes to take effect

🎯 HOW TO USE:
-------------
1. Start speaking: "alexa [your command]"
2. Examples:
   - "alexa what time is it?"
   - "alexa play highway to hell"
   - "alexa open firefox"
   - "alexa word game"
   - "alexa market for google"
   - "alexa send email"
   - "alexa show me" (takes a selfie via camera)

📋 AVAILABLE COMMANDS:
---------------------
  time, date, play YouTube, google search, screenshot
  timer for (X hours/minutes), alarm for (7 a.m./p.m.)
  send email, joke, news, show me (selfie), draw me
  dictionary, bangla, download YouTube video
  Games: tik tak toe, connect computer, connect, word game
  Finance: market for <company>, price of <crypto>
  open <app>, press <key>, type <text>, map <key1 key2>
  instruction (help), stop

🛠️ MANAGEMENT:
--------------
  sudo systemctl status vpa.service    # Check status
  sudo systemctl stop vpa.service      # Stop
  sudo systemctl start vpa.service     # Start
  sudo systemctl restart vpa.service   # Restart
  sudo journalctl -u vpa.service -f    # View logs

❌ TROUBLESHOOTING:
------------------
- No microphone: Check if in 'audio' group
- No camera: Check if in 'video' group
- GUI apps not opening: Make sure you're logged into desktop
- Speech recognition fails: Check internet connection
- Service won't start: Check logs with journalctl

📚 FILES:
--------
  main.py              - Main entry point (clean, modular)
  config.py            - Configuration (emails, apps)
  setup_linux_service.sh  - Automated setup
  run_vpa.sh           - Manual test runner
  verify_setup.py      - Check installation
  vpa.service          - Systemd service file
  README_LINUX.md      - Detailed documentation
  requirements.txt     - Python dependencies
  how_to_get_alarm_sound.sh - Alarm sound helper

💡 NOTES:
-------
- The service runs as your user (not root) for security
- All GUI features need an active desktop session
- Voice recognition uses Google Speech API (requires internet)
- Games and plots open on your current display
- Logs go to systemd journal
