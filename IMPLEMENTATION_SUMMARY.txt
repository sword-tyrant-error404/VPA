🔌 ARCHITECTURE:
---------------
main.py (lightweight coordinator)
  ↓ (imports)
config.py → APPS, WEB_APPS, EMAILS
utils/launcher.py → launch(), launch_and_listen()
utils/tts.py → talk1(), talk2(), tts_worker()
utils/platform.py → open_app(), open_camera(), etc.

main.py spawns subprocesses:
  launch('games/tiktaktoe.py')
  launch('tools/timer.py', 'timer for 5 minutes')
  launch('media/stock.py', 'Apple')
  etc.

Each module is self-contained with its own TTS engine.

🐧 TO RUN ON LINUX:
------------------
Step 1: Test in terminal
   ./run_vpa.sh

Step 2: Verify setup
   python3 verify_setup.py
   (Fix any missing packages shown)

Step 3: Get alarm sound (optional but recommended)
   ./how_to_get_alarm_sound.sh
   or just ignore - timer/alarm will be silent

Step 4: Install systemd service
   sudo ./setup_linux_service.sh
   (Copy the service file to /etc/systemd/system/)

Step 5: Enable and start
   sudo systemctl enable vpa.service
   sudo systemctl start vpa.service

Step 6: Verify it's running
   sudo systemctl status vpa.service
   sudo journalctl -u vpa.service -f

🔐 REQUIRED PERMISSIONS:
-----------------------
Add user to required groups (then logout/login):
   sudo usermod -a -G audio $USER   # Microphone
   sudo usermod -a -G video $USER   # Camera

For GUI access, ensure X11/Wayland is accessible:
   xhost +SI:localuser:$USER

🆘 TROUBLESHOOTING:
------------------
Service won't start:
   → Check logs: sudo journalctl -u vpa.service -n 50
   → Verify permissions (audio/video groups)
   → Test manually: ./run_vpa.sh

No microphone:
   → Check if in 'audio' group: groups
   → Verify mic not muted: alsamixer or pulseaudio volume control

No sound output:
   → Check PulseAudio: pactl list sinks
   → Test speaker: speaker-test -c 2 -t wav

GUI apps fail:
   → Ensure DISPLAY is set: echo $DISPLAY
   → Ensure Xauthority: xhost +SI:localuser:$USER
   → Try from terminal first (./run_vpa.sh)

Speech recognition errors:
   → Check internet connection
   → Verify microphone works in other apps
